# Coke Machine

A coke machine accepts coins: 25 cents, 10 cents, and 5 cents. Prompt the user repeatedly for a coin until they've paid at least 50 cents. After each insertion print the amount due. When paid in full, print how much change is owed.

**Example:**
```
Amount Due: 50
Insert Coin: 25
Amount Due: 25
Insert Coin: 25
Change Owed: 0
```
