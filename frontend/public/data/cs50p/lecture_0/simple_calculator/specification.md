# Specification

Write a program that acts as a simple calculator.

The program should:
1. Ask the user for two numbers
2. Ask the user for an operator (`+`, `-`, `*`, or `/`)
3. Perform the correct mathematical operation
4. Display the result

Example:

Input:
10
+
5

Output:
15

Another Example:

Input:
8
/
2

Output:
4.0
