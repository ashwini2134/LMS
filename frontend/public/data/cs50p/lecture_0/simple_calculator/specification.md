# Specification

Write a Python program that works like a simple calculator.

The program should:
- Ask the user for two numbers
- Ask the user for an arithmetic operator (`+`, `-`, `*`, or `/`)
- Perform the correct calculation
- Print the result

## Example

```text
Input:
10
+
5

Output:
15
```

```text
Input:
8
/
2

Output:
4.0
```
