# Background

Arithmetic operations are one of the most important building blocks in programming. In Python, you can perform mathematical calculations using operators such as `+`, `-`, `*`, `/`, and `**`. The `**` operator is specifically used for exponentiation, which means raising a number to a power.

In this problem, students will practice calculating the square of a number. Squaring means multiplying a number by itself. For example, the square of 5 is 25. This exercise also introduces the difference between integers and floating-point numbers when working with user input and mathematical operations.
