# Lecture 1: Functions, Variables

Welcome to the first lecture of CS50 Python. In this lecture, we'll learn about variables and functions.

## Variables

Variables allow us to store data so that we can use it later.
```python
name = input("What's your name? ")
print(f"hello, {name}")
```

## Functions

Functions are reusable blocks of code. `print` and `input` are built-in functions. You can also define your own!
```python
def hello(to="world"):
    print("hello,", to)
```
