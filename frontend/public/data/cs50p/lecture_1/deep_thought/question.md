# Deep Thought

According to the Hitchhiker's Guide to the Galaxy, the answer to the Great Question of Life, the Universe and Everything is 42. Write a program that prompts the user for the answer to the Great Question. If the answer is 42 (or "forty-two" or "forty two"), output "Yes, that's the answer!". Otherwise output "No".

**Example:**
```
What is the Answer to the Great Question of Life, the Universe, and Everything? 42
Yes, that's the answer!
```
