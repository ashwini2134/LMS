# File Extensions

Prompt the user for the name of a file and output that file's media type based on the extension. Handle: .gif → image/gif, .jpg/.jpeg → image/jpeg, .png → image/png, .pdf → application/pdf, .txt → text/plain, .zip → application/zip. Anything else outputs application/octet-stream.
