# Background

In Python, arithmetic operators allow you to perform mathematical calculations such as addition, subtraction, multiplication, and division. One common operation is finding the square of a number.

Python provides two ways to square a value. You can multiply the number by itself using `number * number`, or use the exponentiation operator `**` like `number ** 2`.

Programs may work with integers or floating-point numbers. Integers are whole numbers, while floats contain decimal values.
